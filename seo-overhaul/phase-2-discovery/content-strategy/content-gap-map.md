**Date:** 2026-03-23
**Phase:** 2 -- Discovery
**Author:** Content Strategist Agent

## How to Read This Map

Each gap is scored on two axes:
- **Commercial Value** (1-5): How directly does this content drive consulting bookings?
- **SERP Opportunity** (1-5): How weak is the current competition for this keyword?

**Priority Score** = Commercial Value x SERP Opportunity (max 25)
